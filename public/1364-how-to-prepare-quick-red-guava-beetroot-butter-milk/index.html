Instructions to make Red Guava Beetroot Butter milk:
  1. Roast beetroot n peel. Chop guava n beetroot. N churn in a mixie jar add 2tb water. N churn again.
  2. Now strain pulp in a strainer. N add in a butter milk add black salt n roasted cumin seeds powder churn again n serve. Healthy tasty butter milk is ready to serve. Garnish glass with beetroot n guava pulp.
